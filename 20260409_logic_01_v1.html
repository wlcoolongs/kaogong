张、王、李、赵四人职位分配推理

题目条件

四人（张、王、李、赵）各负责一项工作：行政管理、销售、研发、安保。

① 张：不做行政管理，不做安保
② 王：不做行政管理，不做研发
③ 如果张没有做研发，那么赵也没有做行政管理
④ 李：不做行政管理，不做安保
⑤ 赵：不做研发，不做安保

Step 1 · 直接排除

赵 → 行政管理

→ 前件必须为假
→ 张没有"没做研发"
→ 张 = 研发 ✓
Step 3 · 确定张
张 → 研发

剩余职位：销售、安保
剩余人员：王、李

由条件④：李 ≠ 安保
李 ≠ 安保 → 李 = 销售
王 = 安保
Step 4 · 验证
逐条验证结论
① 张=研发 ✓（非行政非安保）
② 王=安保 ✓（非行政非研发）
③ 张做研发→前件为假→条件③自动满足 ✓
④ 李=销售 ✓（非行政非安保）
⑤ 赵=行政 ✓（非研发非安保）

全部验证通过 🎉

推理链路

张/王/李 均≠行政 → 赵 = 行政管理 （唯一剩余）
赵 = 行政管理 → 条件③后件为假 → 前件必须为假 → 张 = 研发
张 = 研发 + 李 ≠ 安保（条件④） → 李 = 销售 → 王 = 安保
