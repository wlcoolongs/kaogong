行政管理、销售、研发、安保 —— 条件推理 + 可视化分支法
📋 一、真题

大学毕业的张、王、李、赵4人应聘到了同一家大型公司，每人负责一项工作，其中一人做行政管理，一人做销售，一人做研发，另一人做安保。已知：

① 张不做行政管理，也不做安保；
② 王不做行政管理，也不做研发；
③ 如果张没有做研发，那么赵也没有做行政管理；
④ 李不做行政管理，也不做安保；
⑤ 赵不做研发，也不做安保。

由此可以推出：
A. 张做销售，李做研发
B. 赵做研发，李做销售
C. 李做销售，张做研发
D. 李做研发，赵做安保

✅ 二、答案速览
正确选项 C. 李做销售，张做研发 唯一满足所有条件的分配。
🔍 三、已知条件
①张 ≠ 行政管理，张 ≠ 安保 → 张 ∈ {销售, 研发}
②王 ≠ 行政管理，王 ≠ 研发 → 王 ∈ {销售, 安保}
③如果张没有做研发，那么赵也没有做行政管理 ⇔ 张=研发 或 赵≠行政管理
④李 ≠ 行政管理，李 ≠ 安保 → 李 ∈ {销售, 研发}
⑤赵 ≠ 研发，赵 ≠ 安保 → 赵 ∈ {行政管理, 销售}
🎨 四、可视化推理（分支法 + 表格）
📊 步骤1：根据条件①、②、④、⑤列出每人可能岗位
  • 张：由① → 只能是销售或研发
  • 王：由② → 只能是销售或安保
  • 李：由④ → 只能是销售或研发
  • 赵：由⑤ → 只能是行政管理或销售

✅ 步骤3：唯一解分配表
推理结论： 假设②（张=销售）导致矛盾，因此只有假设①（张=研发）成立。
推导过程：
  • 张=研发 → 剩余岗位：行政、销售、安保
  • 李 ∈ {销售,研发} → 李=销售
  • 王 ∈ {销售,安保} → 王=安保
  • 赵 ∈ {行政,销售} → 赵=行政
得到唯一分配：张→研发，王→安保，李→销售，赵→行政管理。

🎯 步骤4：验证选项
  • A. 张做销售，李做研发 → 与实际（张研发、李销售）不符 ❌
  • B. 赵做研发，李做销售 → 赵实际是行政，不是研发 ❌
  • C. 李做销售，张做研发 → 与实际完全一致 ✅
  • D. 李做研发，赵做安保 → 李实际销售，赵行政 ❌
A. 张做销售，李做研发
B. 赵做研发，李做销售
C. 李做销售，张做研发
D. 李做研发，赵做安保
🌟 因此，正确选项是 C. 李做销售，张做研发 🌟
