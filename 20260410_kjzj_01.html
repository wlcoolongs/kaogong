🎯 初始计量题型 · 三步判断法

📖 典型例题

A公司和B公司为同一集团下的两个子公司。2×19年6月30日，A公司以一项固定资产作为对价取得B公司80%的股权，另支付审计咨询费用80万元。该固定资产原值900万元，已提折旧450万元，当日公允价值700万元。B公司相对于最终控制方而言的可辨认净资产账面价值为900万元，公允价值为1000万元。则A公司取得该项长期股权投资的初始投资成本为多少？

flowchart LR A[属于同一控制下企业合并] --> B[初始投资成本 = B公司账面价值 × 持股比例] B --> C[900万元 × 80% = 720万元] C --> D[审计咨询费80万元 → 管理费用] D --> E[正确答案：720万元]
📌 解析思路：同一控制下合并，初始成本按被合并方账面净资产份额确定，合并直接费用计入当期损益（管理费用）。
flowchart TD Start([开始：取得长投]) --> Type{取得方式} Type -->|同一控制下合并| Same[同一控制下] Type -->|非同一控制下合并| NotSame[非同一控制下] Type -->|非企业合并方式| NoMerge[非合并方式] Same --> S1[初始成本 = 最终控制方账面净资产份额 + 原商誉] S1 --> S2[合并审计等费用 → 管理费用] S2 --> S3[发行权益证券佣金→冲减资本公积] NotSame --> N1[初始成本 = 付出对价的公允价值] N1 --> N2[合并审计等费用 → 管理费用] N2 --> N3[发行证券佣金→冲减资本公积] NoMerge --> M1[初始成本 = 公允对价 + 交易费用] M1 --> M2[交易费用计入初始成本] S3 --> Done([得出初始投资成本]) N3 --> Done M2 --> Done

⚙️ 后续计量 · 权益法“四大变动”

📖 典型例题

甲公司对乙公司的长期股权投资采用权益法核算，乙公司发生的下列各项交易或事项中，将影响甲公司资产负债表“长期股权投资”项目列报金额的有哪些？

A. 收到用于补偿已发生费用的政府补助100万元
B. 宣告分派现金股利1000万元
C. 其他债权投资公允价值增加50万元
D. 其他权益工具投资公允价值减少30万元

flowchart TD A[乙公司的交易或事项] --> B{是否影响乙公司所有者权益总额?} B -->|是| C[影响甲公司长投账面价值] B -->|否| D[不影响甲公司长投账面价值] A -.->|选项A| E[收到政府补助计入当期损益] E -->|影响净利润| F[→ 影响乙公司所有者权益] F --> C A -.->|选项B| G[宣告分派现金股利] G -->|减少留存收益| H[→ 影响乙公司所有者权益] H --> C A -.->|选项C/D| I[其他债权/权益工具投资公允价值变动] I -->|计入其他综合收益| J[→ 影响乙公司所有者权益] J --> C
✅ 答案：A、B、C、D均正确。凡是影响被投资方所有者权益总额的事项，投资方均需按持股比例调整长期股权投资账面价值。
flowchart TD A[初始投资成本调整] --> A1{成本 vs 可辨认净资产份额} A1 -->|成本小于份额| A2[调增长投账面价值
差额计入营业外收入] A1 -->|成本大于份额| A3[不调整，内含商誉] B[被投资方净利润变动] --> B1[确认投资收益] B1 --> B2[需调整净利润：] B2 --> B3[· 投资时点公允与账面差异] B3 --> B4[· 未实现内部交易损益抵销] C[被投资方其他综合收益变动] --> C1[按持股比例确认
其他综合收益] D[被投资方其他权益变动] --> D1[按持股比例确认
资本公积-其他资本公积] E[被投资方宣告现金股利] --> E1[冲减长期股权投资账面价值] A2 & A3 & B4 & C1 & D1 & E1 --> F[影响长期股权投资账面价值]
🔍 关键点：权益法下长投账面随被投资方所有者权益变化而同向调整，净利润需经过“两个调整”（公允口径+内部交易抵销）。
💡 注意：宣告现金股利时，被投资方所有者权益减少，投资方应冲减长投账面价值，同时确认应收股利。

🔄 核算方法转换 · 五条核心路径

📖 典型例题

甲公司2×23年1月1日取得乙公司10%股权，作为交易性金融资产核算，公允价值500万元。2×24年1月1日，甲公司以2500万元购入乙公司70%股权，原10%股权公允价值变为600万元，合计持股80%取得控制权（非同一控制），乙公司当日可辨认净资产公允价值3000万元。计算甲公司个别报表中长期股权投资的初始成本及商誉。

flowchart LR FA[公允价值计量
金融资产 持股小于20%] -->|追加投资| EQ[权益法
持股20%~50%] FA -->|追加投资取得控制| COST[成本法 持股大于50%] EQ -->|追加投资| COST COST -->|处置股权失去控制
但仍具重大影响| EQ COST -->|处置股权且无重大影响| FA EQ -->|处置股权失去重大影响| FA style FA fill:#cfe8fc style EQ fill:#fff2cc style COST fill:#d5e8d4
📌 解析：该例题属于“公允价值计量 → 成本法”路径。个别报表长投初始成本 = 原股权公允价值600万元 + 新增对价2500万元 = 3100万元；商誉 = 3100 - 3000×80% = 700万元。

📌 转换1：公允价值 → 权益法 追加投资

flowchart LR A[原金融资产按公允价值重分类] --> B[差额计入当期损益或留存收益] C[新增投资公允价值] --> D[初始成本 = 原公允价值 + 新增对价] D --> E{比较初始成本与份额} E -->|小于份额| F[调增长投 差额计入营业外收入] E -->|大于份额| G[内含商誉不调整]

📌 转换2：权益法 → 成本法 追加至控制

flowchart LR A[原权益法下账面价值] --> B[新成本 = 原账面 + 新增对价公允价值] C[原确认的其他综合收益/资本公积] --> D[暂不处理，待处置时结转]

📌 转换3：成本法 → 权益法 失去控制 保留重大影响

flowchart TD A[处置部分股权] --> B[确认处置损益] B --> C[剩余股权由成本法转为权益法] C --> D[追溯调整：比较剩余股权原投资成本与取得日可辨认净资产份额] D --> E[负商誉调增长投 并计入留存收益] D --> F[按权益法确认持有期间的损益变动] F --> G[分录：借 长期股权投资 贷 留存收益/投资收益/资本公积]
⚠️ 成本法转权益法是综合题最高频难点，需追溯调整被投资方自取得日至处置日的净利润、其他综合收益变动等。

📉 处置题型 · 成本法 vs 权益法

📖 典型例题

处置采用权益法核算的长期股权投资时，需要将原持有期间记入“资本公积——其他资本公积”科目的金额转出，计入投资收益。（　）

flowchart TD Disp[处置长期股权投资] --> Method{核算方法} Method -->|成本法| CostD[处置价款与账面差额 → 投资收益] Method -->|权益法| EquityD[处置价款与账面差额 → 投资收益] EquityD --> Add1[原资本公积-其他资本公积 → 转入投资收益] EquityD --> Add2[原其他综合收益 → 按规定转入留存收益或当期损益]
✅ 答案：正确。权益法下原计入资本公积（其他资本公积）的金额在处置时应转入投资收益。

💼 综合案例串联 · 多步骤取得+合并调整

📖 典型例题

甲公司2×23年1月1日取得乙公司10%股权，作为交易性金融资产核算，公允价值500万元。2×23年乙公司实现净利润300万元。2×24年1月1日，甲公司以2500万元购入乙公司70%股权，原10%股权公允价值变为600万元，合计持股80%取得控制权（非同一控制），乙公司当日可辨认净资产公允价值3000万元。2×24年乙公司实现净利润500万元，宣告分配现金股利300万元。要求：
① 计算2×24年1月1日个别报表中长期股权投资的初始成本及商誉。
② 计算2×24年年末合并报表中应调整的长期股权投资余额。

flowchart TD subgraph Step1[2×23年1月1日] A[取得乙公司10%股权
交易性金融资产500万元] end subgraph Step2[2×24年1月1日 追加至80%控制] B1[原10%股权公允价600万元重分类] B2[新增70%股权支付2500万元] B3[长投初始成本 = 600+2500 = 3100万元] B4[乙公司可辨认净资产公允3000万×80%=2400万] B5[商誉 = 3100 - 2400 = 700万元] end subgraph Step3[2×24年末 成本法下后续及合并调整] C1[乙公司净利500万, 宣告现金股利300万] C2[成本法确认投资收益 300×80% = 240万] C3[年末合并报表 成本法转权益法调整] C4[调整后长投余额 = 3100 -300×80% +500×80% = 3260万元] end Step1 --> Step2 --> Step3
🌟 答案：① 初始成本3100万元，商誉700万元；② 合并报表调整后长投余额3260万元。公式：初始成本 - 宣告股利×持股比例 + 净利润×持股比例。

flowchart LR A[顺流或逆流交易] --> B[未实现内部损益] B --> C[调整被投资方净利润 = 账面净利润 - 未实现损益×持股比例] C --> D[以后年度实现时加回]
💎 长期股权投资总结口诀：“同控账面非同控公允，权益法下四调整；成本法简单股利收益，转换追溯别记混；减值一律不许转回，处置结转全入损益。”
